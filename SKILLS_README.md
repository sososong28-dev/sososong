# OpenClaw Skills 集合

这是从本地 OpenClaw 工作区同步的 Skills 集合。

## 已包含的 Skills

| Skill | 说明 |
|-------|------|
| apple-notes | Apple 备忘录管理 |
| apple-notes-alternative | Apple 备忘录 (AppleScript 版) |
| apple-reminders | Apple 提醒事项管理 |
| apple-reminders-alternative | Apple 提醒事项 (AppleScript 版) |
| find-skills | Skill 发现工具 |
| git-essentials | Git 基础命令 |
| humanizer | AI 文本人性化 |
| notebooklm | NotebookLM 集成 |
| openclaw-backup | OpenClaw 备份工具 |
| playwright-scraper | Playwright 网页爬虫 |
| self-improving-agent | 自我改进代理 |
| summarize | 内容摘要工具 |
| summarize-alternative | 内容摘要 (替代方案) |
| tavily-search | Tavily AI 搜索 |
| vetter | Skill 安全审查工具 |

## 使用方法

将这些 Skills 复制到你的 OpenClaw 工作区：

```bash
cp -r skills/* ~/.openclaw/workspace/skills/
```
